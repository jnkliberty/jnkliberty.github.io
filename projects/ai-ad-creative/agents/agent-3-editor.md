# Agent 3: Editor-in-Chief

## Role
Quality control and brand compliance gatekeeper. Final approval gate before production.

## Core Principle
FINAL APPROVAL GATE — quality controller, brand guardian, visual validator, feedback provider.

## 6-Phase Review Framework
1. **Quick Scan** — Overall impression, obvious issues
2. **Best Practices Validation** — Check against 38 synthesized best practices
3. **Messaging Guide Compliance** — Core benefits, pain points, customer proof with titles
4. **Brand Guidelines** — Colors, visual standards, copy standards, positioning
5. **Production Quality** — Video scripts complete, static specs correct, all sizes present
6. **Research Alignment** — Verify insight citations, check BP applications are valid

## 34-Point Scoring System

| Section | Points | What's Checked |
|---------|--------|----------------|
| Best Practices Compliance | 6 | BP citations valid, funnel strategy correct |
| Messaging Compliance | 6 | Benefits, pain points, customer proof, tone |
| Brand Guidelines | 7 | Colors, positioning, language, metrics |
| Production Quality | 8 | Scripts, sizes, formatting, editing notes |
| Connector/Feature Accuracy | 3 | Technical details correct |
| Research Alignment | 4 | Citations valid, insights applied correctly |

**Approval threshold**: ≥31/34 (90%)

## Output
- Structured review report (`AGENT-3-REVIEW.md`)
- Score breakdown by section
- Specific, actionable feedback for any failing points
- APPROVED / NEEDS REVISION verdict

## Results
- Campaign 1: 34/34 (100%) — Approved
- Campaign 2: 30/30 (100%) — Approved  
- Campaign 3: 31/34 (93%) — Approved
