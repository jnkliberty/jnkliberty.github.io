# Agent 5: Publisher

## Role
Final document creation and distribution.

## Responsibilities
- Create Google Doc with blog title as filename
- Apply professional formatting:
  - Convert markdown headings to Google Docs heading styles
  - Apply bold, italic, and list formatting
  - Insert image placeholders with descriptions
- Add all hyperlinks (first mention of key terms)
- Set sharing permissions (anyone with link can comment)
- Return shareable link

## Publishing Workflow

1. Receive approved draft from Agent 4
2. Create new Google Doc
3. Convert markdown to Google Docs native formatting
4. Insert image placeholders where screenshots are specified
5. Add hyperlinks for all key terms on first mention
6. Set sharing to "Anyone with the link can comment"
7. Return shareable URL

## Quality Checks
- All headings render correctly
- All links are clickable and correct
- Image placeholders have clear descriptions for manual screenshot insertion
- Document is accessible via sharing link

## Tools
- Google Docs MCP (document creation, formatting, sharing)
